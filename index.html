Some of my recent work.

OHI/O – The branding site for the OHI/O hackathon at Ohio State. It's built on AngularJS and uses SheetRock to pull all of the live content from Google Sheets. This lets any member edit content, regardless of web experience.

John Patrick Halling – Promo page for musician from Kent, OH. Uses the WP-API to let John manage everything through Wordpress. The Rest API lets us host on Github pages, getting away from WP themes and Ads.

GeoPatternsCocoa – Port of Jason Long's Ruby Library for iOS and OS X applications. By using CoreGraphics, it can render faster than wrapping and displaying an SVG.

MyPharmasist – Capstone iOS application to manage your prescriptions and learn more about the Ohio State Pharmacy. *Waiting on OSU approval*

Outside of full projects, I open source some reusable components. Some examples includes a better UITextField, a silly little npm module, iPhone step counter, as well as some contributions to others' projects
