Xero Me - Xero Me is for employees and payroll admins of businesses who use Xero to manage payroll. Xero Me allows employees to check your pay history, submit timesheets and manage leave requests from anywhere. Payroll admins can approve leave requests, timesheets and view details of employees.

Spotify Today - This is an OS X widget for Spotify. It lets you control playback, save songs, and share songs, all from the Today Widget pane! It's all in Swift.

ReactPants - Sometimes when you have something really important to do, you program instead. ReactPants provides Spongebob-based gifs for the emotion you search for. It's build using Reactive Programming, which is really cool stuff. You can add more emotions by provide a URL and a keyword list to the JSON file.

JohnPatrickhalling - Promo page for musician from Kent, OH. Uses the WP-API to let John manage everything through Wordpress. The Rest API lets us host on Github pages, getting away from WP themes and Ads.

GeoPatterns - Port of Jason Long's Ruby Library for iOS and OS X applications. By using CoreGraphics, it can render faster than wrapping and displaying an SVG.
